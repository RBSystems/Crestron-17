HasChildren
Contains a flag indicating if node has children nodes in the TreeNodeChildren property. This property is not required and has a default value of false.
LastModified
Indicates date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
MajorID
MajorID
NodeID
Identifies the node using a globally unique identifier (GUID). This property is read only. The asset id should be placed in the URI for PUT and DELETE operations. The default is a new GUID. The maximum length is 50 characters.
NodeName
Contains the display name for the node This field is required and has a maximum length of 50 characters.
NodePath
Path of node
NodeType
Indicates the type of node using the API_Enum_NodeType enumeration. This property is optional with a default value of 'None'.
ParentNodeID
Stores the globally unique identifier (GUID) of the parent node of the node being represented. This property is required and has a maximum length of 50.
TreeNodeChildren
Contains a collection of Tree Nodes which are children of the node being represented. This property is optional and has default value of null.
TreeNodeRooms
Contains a collection of Rooms which are children of the node being represented. This property is optional and has default value of null.
