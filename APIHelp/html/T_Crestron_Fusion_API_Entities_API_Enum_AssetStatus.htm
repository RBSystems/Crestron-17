            Indicates the current date is past the estimated lifespan of the asset.
            This is set when the DateOfPurchase and the LifeSpanYears
            properties of the API_Asset class exceed the current date

            Indicates that the last significant event of an asset was a change in the Node or Room which the asset is associated.
