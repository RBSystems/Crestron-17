Properties
  NameDescription
ActionDescription
Consists of a string representing a description for the action. This property is optional with a default of null.
ActionID
Identifies the action using a globally unique identifier (GUID). This property is read only. The asset id should be placed in the URI for PUT and DELETE operations. The default is a new GUID. The maximum length is 50 characters.
ActionName
Consists of a string representing the name of the action. This property is required for POST and PUT operations. The maximum length is 128 characters.
IsOverride
Indicates if an action is overridden. This property is optional with a default of false.
LastModified
Contains date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
OffsetMinutes
Indicates the numer of minutes to wait for action to start. This property is not required and has a default of 0 The wait interval begins at the end of the preceding action or at the start of the seqeunce if action is in first ordinal position.
OverriddenActionID
Indicates if the action has been overridden by another action. This property is optional and has a default value of null.
StepList
Consists of a collection of action steps comprising the action. This property is optional and has a default value of null.
