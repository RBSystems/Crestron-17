Crestron.Fusion.APICrestron Fusion API Online Help
Welcome to the Fusion API. Please note the following:
  • XML elements submitted to server must be in alphabetical order within each node.
  • Json enumerations must be submitted as the enumeration value rather than string label.
  • Json DateTime values are represented as milliseconds after 1/1/1970. The timestamp is in UTC timezone with an optional offset to convert to local time on server.
Namespaces
NamespaceDescription
Crestron.Fusion.API.ApiHelper
Contains functions useful to client applications.
Crestron.Fusion.API.Entities
Contains entity classes used by Fusion API Server, which can be shared with client applications.
Crestron.Fusion.API.Resources
Contains classes which respond to REST requests over HTTP protocol. URI Details are specified on the summary page of each class. Data can be exchanged in either JSON or XML (default) format depending on the value specified in the Content-Type http header variable.
Crestron.Fusion.API.Security
Contains service endpoints for user authorization and security token generation.
