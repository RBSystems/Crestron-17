GroupwareProviderType
Contains the type of groupware provider for the room. This property is required. The maximum length of the string is 10 characters.

The value must be one of the following:
  • None
    No provider--can only do LDAP and non-specific stuff
  • Exchange
    MS Exchange (WebDAV) [up to Exchange 2007]
  • EWS
    MS Exchange Web Services [Exchange 2007 and up]
  • Internal
    Internal scheduler (v 7.2.0010 and up)
  • Domino
    Lotus Domino (v 7.2.0010 and up)
  • Micros
    Micros Opera (v 8.1 and up)
  • R25
    College Net R25
  • External
    Custom Scheduling Provider
  • Google
    Google Calendars

TimeZoneID
Contains an identifier representing the timezone where the room is located. The timezone identifiers are the key names available in the Fusion web client server registry at location HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\Time Zones (i.e. "Pacific Standard Time"). The identifiers can also be accessed using the .net method System.TimeZoneInfo.GetSystemTimeZones() This property is required with a maximum length is 255 characters.
