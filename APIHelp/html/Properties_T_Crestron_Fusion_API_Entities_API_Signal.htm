API_Signal PropertiesCrestron Fusion API Online Help
The API_Signal type exposes the following members.
Properties
  NameDescription
AttributeID
Indicates the unique identifier of the attribute associated with this symbol. This property is required with a maximum length of 50 characters.
AttributeName
Consists of a string representing the name of the attribute associated with the signal. This property is optional. The maximum length is 255 characters.
AttributeType
Indicates the type of attribute using the API_Enum_AttributeType enumeration. This property is required.
DefaultIOMask
Indicates the type of IO mask using the API_Enum_AttributeType enumeration. This property is required.
JoinNumber
Identifies the Symbol Attribute association. This value is optional with a default of null.
LastModified
Indicates date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
LogicalOperator
Indicates the type of logical operator for the signal using the API_Enum_LogicalOperator enumeration. This property is required.
RawValue
Consists of a string representing raw value of a signal. This property is not required and has a default value of null.
Reserved
Indicates the signal is read only and cannot be overridden. This property is optional with a default of null.
SignalID
Identifies the signal using a globally unique identifier (GUID). This property is read only. The asset id should be placed in the URI for PUT and DELETE operations. The default is a new GUID. The maximum length is 50 characters.
SignalMaxValue
Indicates the maximum value a signal is capable of containing. This property is not required and has a default value of null.
SignalMinValue
Indicates the minimum value a signal is capable of containing. This property is not required and has a default value of null.
SignalName
Consists of a string representing the signal's name. The maximum length is 255 characters.
Slot
Identifies the symbol relative to other signals. This property is optional with a default of null.
SymbolID
Indicates ID of symbol associated with the signal as represented in API_Signal. This field is not required and has a default of null.
SymbolName
Indicates name of symbol associated with the signal as represented in API_Signal . This field is not required and has a default of null.
XmlName
Indicates name the of symbol associated with the signal as represented in API_Signal . This field is not required and has a default of null.
