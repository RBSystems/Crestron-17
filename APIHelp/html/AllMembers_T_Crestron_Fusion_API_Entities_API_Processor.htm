Properties
  NameDescription
Autodiscover
Indicates if server will automatically discover symbols associated with processor. This property is optional with a default of false.
Connected
Indicates if processor is connected to a network. This property is optional with a default of false.
ConnectInfo
Contains the IP address or hostname of processor. This property is optional with a default of null.
ConnectSSL
Inidcates if connections made by processor are to use Secure Sockets Layer. This property is optional with a default of null.
IPID
Contains a string indicating the Crestron IP identifier (IPID) of the asset. This property is optional with a default value of 0.
LastModified
Indicates date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
Location
Contains a test string describing the location of a processor This property is optional with a default value of null. The maximum length is 128 characters.
ParentID
Contains a globally unique identifier (GUID) indicating the parent node containing the processor. This property is optional with a default of null.
Password
The password associated with the username property. This property is optional with a default of null. The maximum length is 350 characters.
Port
The port number of the processor. This property is required and is of integer type.
ProcessorID
Identifies the processor using a globally unique identifier (GUID). This property is optional with a default value if null. The maximum length is 50 characters.
ProcessorName
Consists of a string representing the processor's name. The maximum length is 128 characters.
SecurePort
Indicates port number to be used by secure sockets layer. This property is optional with a default of null.
Symbols
Contains a collection of symbols associated with the room. This property is optional with a default value of null.
Username
Consists of a string representing the processor user's name. The maximum length is 34 characters.
