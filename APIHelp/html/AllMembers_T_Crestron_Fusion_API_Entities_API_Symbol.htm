Properties
  NameDescription
ConnectInfo
Contains the IP Address for the asset symbol is attahced. This property is required for POST and optional on PUT with a default of null.
IPID
Contains a string indicating the Crestron IP identifier (IPID) of the asset. This property is optional with a default value of 0.
LastModified
Indicates date and time the record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
NodeID
Indicates the API_Node associated with the signal. This property is optional with a default of null.
NodeText
Indicates the API_Node name associated with the signal. This property is optional with a default of null.
Password
Contains password associated with symbol. This property is optional with a default of null.
Port
The port number of the processor. This property is optional and is of integer type.
ProcessorID
Indicates the unique id of the processor associated with the room. This property is optional with a default of null. The maximum length of the column is 50 characters.
ProcessorName
Stores the name of the processor associated with the room. This property is optional with a default value of null.
RoomID
Contains a room or node identifier (GUID) for the symbol. This property is optional. Maximum length is 50 characters.
SecurePort
Indicates port number to be used by secure sockets layer. This property is optional with a default of null.
Signals
Contains a collection of API_Signal types representing signals associated to the symbol. This property is optional and has a default value of null.
SymbolID
Identifies the symbol using a globally unique identifier (GUID). This property is read only. The asset id should be placed in the URI for PUT and DELETE operations. The default is a new GUID. The maximum length is 50 characters.
SymbolName
Consists of a string representing the symbol's name. This property is required for POST and PUT operations. The maximum length is 255 characters.
UserName
Indicates the name of user associated with the symbol. This property is optional.
UseSSL
Indicates the the symbol should use secure sockets layer for data communication. This property is optional.
Version
Version of symbol. Currently the possible values are 6,7,8. This property is optional with a default of null.
