Properties
  NameDescription
DisplayName
Consists of the location name displayed in the web client. This property is required and has a maximum length of 128 characters.
EntryIsEmpty
Indicates that object has a null value. This property is optional with a default of null.
LastModified
Indicates date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
LocationTypeID
Stores the location type id as displayed in the Fusion web client. This property is required and has a maximum length of 15 characters.
ManualEntry
Indicates that the location information was manually assigned as opposed to inheirited from parent nodes. This value is required.
NodeID
Indicates the globally unique identifier for the node represented by the location. This property is optional with a default of null.
NodeText
Indicates the name of node represented by the location. This property is optional with a default of null.
ParentLocationTypeID
Indicates the location type of the node parent. This property is optional with a default of null.
Value
Contains the name of the location. This property is required.
