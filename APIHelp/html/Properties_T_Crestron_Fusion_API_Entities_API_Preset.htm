API_Preset PropertiesCrestron Fusion API Online Help
The API_Preset type exposes the following members.
Properties
  NameDescription
Enabled
Indicates if the preset is enabled. This property is required and has a default value of false.
ImageSource1
Contains path to first image used by preset. This property is optional with a default value of null.
ImageSource2
Contains path to second image used by preset. This property is optional with a default value of null.
LastModified
Contains date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
MaximumPresets
Contains a read only value indicating the maximum number of presets that can be used. This is not required and has a default of 10.
PresetFields
Contains a collection of API_PresetField types. This property is optional and has a default of null.
PresetName
Contains the name of the preset as shown in the fusion web client. This property is required and has maximum length of 50 characters.
PresetSlot
Stores the slot number associated with the preset. This property optional with a default value of -1.
WelcomeMessage
Contains the welcome message for the preset. This field is not required and has a default value of null.
