Message
Contains the message describing a fault, if a failure occurred on the resource operation request.
PageNumber
Represents the page number of the current result set. This property is used on read many operations where the results are split into pages of 25 rows. The page number can be set for read many operation requests by setting a value to the "page" query string variable (?page=).
Source
Contains the namespace and class name where a fault occured, if a failure occured on the resource operation request.
StackTrace
Contains a string representation of the immediate frames in the call stack where a fault occured, if a failure occured on the resource operation request.
Status
Contains a word or a phrase describing an error, which correlates to the http status code returned by the request. This value is populated on both successful and faulted resource operation reuquests.
TotalRecords
Repesents the total count of records returned from read many operation without regard to paging. This property is used to determine if other pages of results are availible.
