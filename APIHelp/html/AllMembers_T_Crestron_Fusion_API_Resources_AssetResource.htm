Methods
  NameDescription
DeleteAsset
Requests for the delete operation on the asset resource are routed to this method.
GetAsset
Requests for the read one operation on the asset resource are routed to this method. The WCF Rest service contract will automatically serialize the API_Result object into outbound XML or Json
GetAssets
Requests for the read many operation on the asset resource are routed to this method. The following parameters are sent in the request URI query string to filter the results

?search=Search string on Asset name

?page=the page number of the result set. Each page except last is 25 rows

The WCF Rest service contract will automatically serialize the API_Result object into outbound XML or Json
PostAsset
Requests for the create operation on the asset resource are routed to this method. The WCF Rest service contract will automatically deserialize Inbound XML or Json into an API_Asset object and serialize the API_Result object into outbound XML or Json
PutAsset
Requests for the update operation on the asset resource are routed to this method. The WCF rest service contract will automatically serialize the API_Result object into outbound XML or Json
