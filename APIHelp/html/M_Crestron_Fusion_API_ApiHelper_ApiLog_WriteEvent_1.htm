Remarks

            If this is called, we always write the message to the Event Log, assuming it's available.
            We write to the text log unless the Log Level is NONE, so this is effectively the same as
            WriteNormal(), with the bonus of going to the Event Log in all cases.  Note: this 
            cookie-cutter nonsense is done with the override because we need to be on the same level
            in the call stack to print the proper originating function.
