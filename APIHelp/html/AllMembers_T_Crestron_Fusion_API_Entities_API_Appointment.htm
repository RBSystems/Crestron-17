Represents the External Provider's meeting identifier; it may be required, depending on the groupware application. If not, it MUST be set to the same value as the GW_MeetingID field.

Represents the time the meeting ends (optional).
EventOrMeeting
This is an enum field that identifies the type of appointment (optional). The possible values are defined by the API_Enum_EventOrMeeting enumeration. The default value is None.
GW_MeetingID
This represents the unique meeting ID that is assigned by the groupware application. This is a required field.
IsPrivate
Identifies whether this meeting has been designated as private by its creator (optional).
LastModified
Indicates the time stamp for the last time the object was modified (optional).
Location
Indicates the location where the meeting takes place (optional). This is a free-form text field with a maximum length of 250 characters.
MeetingComment
Contains the body of the message which describes the meeting (optional). E.g. this may contain the agenda. This may contain either plain text or HTML, depending on the calendaring application.
MeetingSubject
Indicates the subject for the meeting (required). The maximum length is 255 characters.
MeetingType
Identifies the type of the meeting with regard to recurrence. The possible values are defined by the API_Enum_MeetingType enumeration.
NotifyAction
Indicates the type of action which a push notification has generated (optional). Note that it is unusual for this to be provided from outside of the Fusion services, so typically this field is either unused or else should be set to None. The possible values are defined by the API_Enum_PushNotifyAction enumeration.
Organizer
Identifies the meeting organizer (optional). The maximum length of this field is 255 characters.
Preset
Contains a meeting preset which identifies parameters to be sent to the room's processor before the start of the meeting (optional).
RolesForMeeting
Contains a list of user roles assigned to this meeting (optional).
RoomID
Contains a string of comma separated unique room identifiers (GUIDs), that link the meeting to one or more rooms. This field is required on posts and optional on puts.
RRule
For recurring meetings, this field contains the iCal (RFC 5545) recurrence rule. For single-instance meetings, set this field to an empty string. This field is optional, and if not present, a single-instance meeting is assumed.
RV_MeetingID
Contains the Fusion (RoomView) internal meeting GUID. This field is read-only.
Start
Indicates the start time for the meeting. This property is required on post operations.
TimeZoneId
The Windows time zone identifier.
