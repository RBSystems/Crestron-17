Properties
  NameDescription
AssetCustomFields
Contains a collection of custom fields (API_AssetCustomField) associated with the asset. This property is optional with a default of null.
AssetID
Identifies the asset using a globally unique identifier (GUID). This property is read only. The asset id should be placed in the URI for PUT and DELETE operations. The default is a new GUID. The maximum length is 50 characters.
AssetName
Consists of a string representing the asset's name. This property is required for POST and PUT operations. The maximum length is 255 characters.
AssetTag
Consists of tag for the asset. This property is optional with a default value of null. The maximum length for the string is 255.
AssetTypeID
Indicates the asset type identifier. This property is required on POST and PUT operations. The maximum length is 50 characters.
AssetTypeName
ConnType
Indicates the connection type for the asset using the API_Enum_AssetConnectionType enumeration. This property is optional with a default value of 'None'.
DateOfPurchase
Stores the date the asset was purchased. This property is optional with a default value of the current date.
DriverID
Contains a string indicating the driver id for the asset. This property is optional with a default value of null. The maximum Length is 50 characters
IPAddress
Contains a string indicating the internet protocol address of the asset. This property is optional with a default value of null. The maximum length 255.
IPID
Contains a string indicating the Crestron IP identifier (IPID) of the asset. This property is optional with a default value of 0.
LastModified
Indicates date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
LastService
Contains a date and time indicating when the asset was last serviced. This property is optional with a default of the current date and time.
LifeSpanYears
Contains a numeric value representing the life span of the asset. This property is optional. The default value is available from the setup web client.
MACAddress
Stores a string containing the MAC address of the asset. This property is optional with a default of null. The maximum Length is 50 characters.
MaintenanceContractID
Contains a string indicating the identifier for the maintenance contract. This property is optional with a default of null. The maximum Length is 50 characters.
Make
Indicates the make of an asset. This property is optional with a default of null. The maximum length of the string is 255 characters.
Model
Indicates the model of an asset. This property is optional with a default of null. The maximum length of the string is 255 characters.
Notes
Stores a string containing notes for an asset. This property is optional with a default of null.
Password
Contains a string representing the password for the asset. This property is optional with a default of null. The maximum length is 255 characters.
Port
Contains a numeric value holding the Port number for the asset. This property is optional with a default value of -1.
RoomID
Contains a room or node identifier (GUID) for the asset. This property is required. Maximum length is 50 characters.
SerialNumber
Indicates the serial number of the asset. This property is optional with a default of null. The maximum length of the string is 255 characters.
ServiceInterval
Contains a numeric value representing the service interval for the asset. This property is optional. The default value is available from the setup web client.
ServiceIntervalIncrement
Contains a string representing the service interval increment for the asset. This property is optional. The default value is available from the setup web client. The maximum length is 50 characters.
SSL
Stores a boolean value indicating if secure sockets functionality is activated for the asset. This property is optional with a default of false.
Status
Indicates the status for the asset using the API_Enum_AssetStatus enumeration. This property is optional with a default value of 'OK'.
WarrantyExpiration
Gets or sets the date the asset warranty expires. This property is not required and has a default of the current date
