Remarks
(1) Only public properties are returned.
(2) Arrays, classes, and structs are recursively presented.
(3) The specified check (new in 8.1) is used primarily for EWS, since it indicates whether value types
                are present in an object by a bool ...Specified field.  E.g. if there was an int Counter field in the
                object, there would be a CounterSpecified field to indicate whether there was an actual value set
                in the int field on the EWS side.
