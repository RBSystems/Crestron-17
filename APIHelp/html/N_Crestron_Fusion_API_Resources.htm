Crestron.Fusion.API.Resources NamespaceCrestron Fusion API Online Help
Contains classes which respond to REST requests over HTTP protocol. URI Details are specified on the summary page of each class. Data can be exchanged in either JSON or XML (default) format depending on the value specified in the Content-Type http header variable.
Classes
  ClassDescription
ActionResource
Contains logic to respond to action resource operation requests. The operations available include create, read, update, and delete. A send operation is also available which will trigger an action in an asset or a room.

AppointmentResource
Contains logic to respond to appointment resource operation requests. The operations available consist of create, read, update, and delete.
AssetResource
Contains logic to respond to asset resource operation requests. The operations available consist of create, read, update, and delete.
AttributeResource
Contains logic to respond to attribute resource operation requests. The operations available include read, update, and send. The available send operation will trigger an attribute in an asset or a room.

AvailabilityResource
NodeResource
Contains logic to respond to node resource operation requests. The operations available consist of read.
RoomResource
Contains logic to respond to room resource operation requests. The operations available consist of create, read, update, and delete.
SignalValueResource
Contains logic to respond to SignalValue resource operation requests. The operations available consist of read.
