API_Attribute PropertiesCrestron Fusion API Online Help
The API_Attribute type exposes the following members.
Properties
  NameDescription
AssetID
Identifies the unique identifier of the asset associated with the attribute. This field is not required and has a default of null.
AttributeID
Identifies the attribute using a globally unique identifier (GUID). This property is read only. The asset id should be placed in the URI for PUT and DELETE operations. The default is a new GUID. The maximum length is 50 characters.
AttributeName
Consists of a string representing the attribute's name. This property is required for POST and PUT operations. The maximum length is 255 characters.
AttributeType
Indicates the type of attribute using the API_Enum_AttributeType enumeration. This property is required.
ButtonChangeText
Stores the text for the change button shown in the Fusion web client. This property is optional and has a default of null. the maximum length for the string is 128 characters.
ButtonOffText
Stores the text for the off button shown in the Fusion web client. This property is optional and has a default of null. the maximum length for the string is 128 characters.
ButtonOnText
Stores the text for the on button shown in the Fusion web client. This property is optional and has a default of null. the maximum length for the string is 128 characters.
Default
Indicates if the default property values should be used for this attribute. This property is optional with a default of false
DefaultIOMask
Indicates the ability to perform operations consisting of reading and writing data. This property will be assigned to the signal then the attribute is associated with a symbol. This property is required and has a maximum length of 4 characters.
DefaultJoin
Indicates join number to assigned to the signal when the attribute is associated with a symbol. This property is required and has a maximum length of 4 characters.
DefaultMaxValue
Indicates the maximum value of the signal when the attribute is associated with a symbol. This property is optional and has a maximum length of 4 characters.
DefaultMinValue
Indicates the minimum value of the signal when the attribute is associated with a symbol. This property is optional and has a maximum length of 4 characters.
DefaultSlot
Indicates the slot number of the signal when the attribute is associated with a symbol. This property is required and has a maximum length of 10 characters.
GraphicSetID
Stores the unique identifier of the graphics set associated with symbol. This property is not required and has maximum length of 50 characters.
LastModified
Indicates date and time record was last modified. This property is optional on posts and required on puts with a default value the current date. This property is used for record locking. Put requests will need to send the existing value.
LogDB
Stores a boolean flag to indicate if logging information should be stored in the database. This property is optional with a default value of false.
LogNTEvent
Stores a boolean flag to indicate if logging information should be stored in the windows event log. This property is optional with a default value of false.
Reserved
Indicates if attribute is reserved and cannot be overridden. This property is required.
XmlName
Stores the name of the attribute as it should be displayed in web client. This field is required and has a maximum length of 128 characters.
