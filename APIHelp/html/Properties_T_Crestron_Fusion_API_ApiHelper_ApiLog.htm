ApiLog PropertiesCrestron Fusion API Online Help
The ApiLog type exposes the following members.
Properties
  NameDescription
Capacity
Get the current capacity of the cached message list.
Count
Get the current number of saved messages in the cached message list.
FlushSeconds
(R/W) Get or set the number of seconds after which we try to flush the log lines to disk.
FullPathName
(R/O) Get the full path name for the log.
IsActive
(R/O) Is the log currently active?
LogLevel
Get or set the verbosity level of the Log object.
LogsToKeep
(R/W) Get or set the number of logs to keep. Must be >0. (New in 181)
MaxCachedLines
Get or set the maximum number of log lines kept in the cached message list. If this limit is exceeded, we dump the lines to disk at the next timer pop, regardless of whether the interval has expired.
Name
(R/O) Get just the 'filename.ext' of the log file.
Path
(R/O) Get the directory name where logs reside.
ProcessInfo
(R/O) Get footprint info.
WriteToConsole
When true, all output does a Console.WriteLine() as well as the file write. Useful for command-line utilities.
